# FileHubOrg

**FileHubOrg** is a secure, modular, and extensible file-sharing web application designed for internal organizational use. Built with ASP.NET Core and Entity Framework Core, it allows authenticated users to upload, share, and access files based on department, role, and permission level.

---

## 🚀 Features

- 🔐 **Active Directory Integration** for seamless Windows Authentication
- 🗃️ **Role-Based Access Control** (user, department manager, admin)
- 📁 **User-Specific and Departmental File Repositories**
- 💬 **Send Files to Coworkers** with share tracking
- 🧭 **Audit Logging** for upload, delete, view, and share actions
- 🌐 Web-based UI built with Razor Pages / MVC and Bootstrap
- 📎 Configurable file size/type limits
- 📦 Structured, scalable architecture ready for production

---

## 🧰 Technology Stack

- [ASP.NET Core 8.0](https://learn.microsoft.com/en-us/aspnet/core/)
- [Entity Framework Core](https://learn.microsoft.com/en-us/ef/core/)
- SQL Server (or PostgreSQL, SQLite, etc.)
- Identity + Windows Authentication
- Razor Pages or MVC
- Bootstrap 5
